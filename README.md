# CeresFS



## 配置

| 配置项                      | 说明            | 缺省值          |
| ------------------------ | ------------- | ------------ |
| ceresfs.zookeeper.quorum | Zookeeper连接地址 | 无            |
| ceresfs.directories      | 分号分割的图片路径地址   | /tmp/ceresfs |
| ceresfs.vnode.factor     | 虚拟节点倍数        | 1000         |
